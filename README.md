# Drift

A cyberpunk-themed AI search engine powered by Google Gemini 2.5 Flash with search grounding. Enter a query and receive results as a "digital narrative" with a Tron-inspired visual style.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set your Gemini API key:
   ```bash
   export GEMINI_API_KEY=your_api_key_here
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
